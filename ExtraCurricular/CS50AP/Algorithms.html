Formal Reasoning

One form of reasoning is "Proof by Induction"

Proof by Induction
Don't prove everything is true. Just proof enough to be sure
PROCEDURE calcFactorial(n) {
    factorial ← 1
    i ← 1
    REPEAT UNTIL (i > n) {
        factorial ← factorial * i
        i ← i + 1
    }
    RETURN factorial
}
  • Algorithm works for single int
